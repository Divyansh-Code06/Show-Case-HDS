AI Harassment Detection System

Real-time AI-powered detection using YOLOv8 Pose Estimation

Overview

This system uses live webcam feed and YOLOv8 pose estimation to identify aggressive behavior such as potential harassment gestures. It alerts users via an alarm and sends email notifications in real-time, providing a proactive approach to ensure safety.

Key Features

  • Live webcam video analysis
  • YOLOv8 Pose Estimation for gesture detection
  • Alarm trigger on detection
  • Email alert system
  • Simple, fast, and customizable

How to Use

  1. Clone this repository:
    2. git clone https://github.com/Divyansh-Code06/Harassment-Detection-System.git
  2. Install dependencies:
    3. pip install opencv-python numpy ultralytics simpleaudio
  3. Download yolov8n-pose.pt model and place it in the project folder.
  4. Add alarm.wav in the project folder.
  5. Run the detection script:
    6. python harassment_detection_mvp.py
