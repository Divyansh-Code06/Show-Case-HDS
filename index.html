Short Overview

Overview

Our system actively monitors webcam feed to detect gestures related to potential harassment using YOLOv8 pose estimation. It promptly triggers an alarm and sends an email alert, offering immediate response and improved safety.

Key Features

  • Real-time webcam feed analysis
  • YOLOv8 Pose Estimation for gesture detection
  • Automatic alarm activation on suspicious detection
  • Email alerts to authorities for immediate response
  • Modular, lightweight, and easy to deploy

How to Use

  1. Clone the repository:
    2. git clone https://github.com/Divyansh-Code06/Harassment-Detection-System.git
  2. Install required dependencies:
    3. pip install opencv-python numpy ultralytics simpleaudio
  3. Place yolov8n-pose.pt in the root directory.
  4. Add alarm.wav for audio alerts.
  5. Run the detection script:
    6. python harassment_detection_mvp.py
